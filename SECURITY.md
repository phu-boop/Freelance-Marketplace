# Security Policy

## Supported Versions

The following versions of the Freelance Marketplace project are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in the Freelance Marketplace, please report it to us as described below.

**Do not report security vulnerabilities through public GitHub issues.**

### How to Report

Please send an email to security@freelancemarketplace.example.com with the following details:

- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

You should receive a response within 48 hours. We will work with you to verify and resolve the issue.

### Disclosure Policy

- We ask that you give us a reasonable amount of time to fix the issue before making it public.
- We will notify you when the fix is released.
